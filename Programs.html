Google Chrome — самый популярный браузер. Быстрый, много расширений, синхронизация с телефоном.

Mozilla Firefox — отличный выбор для тех, кто ценит приватность и безопасность. Работает на собственном движке.

LibreOffice — мощный бесплатный аналог Microsoft Office. Включает редакторы для текстов, таблиц, презентаций и даже баз данных.

CCleaner — утилита для очистки системы от мусора, кэша браузеров и исправления ошибок реестра.

7-Zip — бесплатный архиватор, который открывает любые форматы (RAR, ZIP, 7z). Незаменимая вещь.

GIMP — бесплатный графический редактор, который может заменить Photoshop для большинства задач (обработка фото, рисование).

Paint.NET — более простая и быстрая альтернатива GIMP для базовых задач: обрезка фото, коррекция цвета, коллажи.

Visual Studio Code (VS Code) — стандарт индустрии на сегодняшний день. Легкий, бесплатный, невероятно мощный за счет магазина расширений. Работает с Git "из коробки".

WebStorm — редактор от JetBrains. Платный, но очень «умный». Глубже понимает код, рефакторинг и автодополнение на голову выше, чем в VS Code.

Sublime Text — "молния" среди редакторов. Открывает огромные файлы за секунды. Очень быстрый и минималистичный.

Krita — это профессиональный графический редактор с открытым исходным кодом, предназначенный в первую очередь для цифровой живописи и 2D-анимации.
